## Context

README.md перечисляет 4 страны (Россия, Украина, Беларусь, Чехия) без разделения на "поддержка номеров" и "только язык". Испания существует в UI как язык интерфейса, но номера не распознаются. Планы развития и структура проекта в README устарели.

## Goals / Non-Goals

**Goals:**
- Актуализировать список стран, разделив их по типу поддержки
- Обновить секцию планов развития
- Обновить структуру проекта, добавив пропущенные файлы
- Уточнить версии в технологическом стеке

**Non-Goals:**
- Не менять код приложения
- Не добавлять/удалять функциональность
- Не менять формальные описания (README остаётся на русском)

## Decisions

- Единственный изменяемый файл — `README.md`
- README написан на русском — продолжать на нём же
- Структура проекта — синхронизировать с актуальным состоянием файловой системы
- Планы развития — отметить Испанию: `[~] Испания: перевод интерфейса (без данных регионов)`

## Risks / Trade-offs

- Никаких рисков — документационное изменение без влияния на код и поведение
