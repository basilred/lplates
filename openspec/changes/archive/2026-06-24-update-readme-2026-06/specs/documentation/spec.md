## ADDED Requirements

### Requirement: Страны с поддержкой номеров

README SHALL перечислять только страны, для которых реализовано распознавание регионов: Россия, Украина, Беларусь, Чехия. Страны, доступные только как язык интерфейса (Испания), SHALL быть указаны отдельно.

#### Scenario: Список стран в README соответствует коду

- **WHEN** пользователь читает раздел "Основные возможности"
- **THEN** страны с поддержкой номеров и страны с переводом интерфейса перечислены раздельно

### Requirement: Актуальные планы развития

README SHALL отражать реальный статус каждой фичи в секции "Планы по развитию".

#### Scenario: Статус Испании в планах

- **WHEN** пользователь читает секцию "Планы по развитию"
- **THEN** Испания отмечена как частично реализованная (только перевод интерфейса)

### Requirement: Полная структура проекта

README SHALL включать все актуальные директории и файлы проекта, включая LPR_ARCHITECTURE.md, scripts/ и .claude/.

#### Scenario: Структура синхронизирована с файловой системой

- **WHEN** пользователь читает секцию "Структура проекта"
- **THEN** перечислены все значимые директории верхнего уровня и корневые файлы
