# LPlates (License Plates)

Современное и быстрое PWA-приложение на React для определения региона по автомобильному номеру или коду региона.

Приложение спроектировано как высокопроизводительный инструмент с премиальным дизайном, поддерживающий автоматический парсинг полных номеров и мультиязычный интерфейс.

## Основные возможности

- **Умный парсинг**: Введите просто код региона (например, `77`) или полный номер автомобиля (например, `А123БВ777` или `AA1234BB`). Приложение само извлечет нужную информацию.
- **Поддержка нескольких стран**:
  - 🇷🇺 Россия
  - 🇺🇦 Украина
  - 🇧🇾 Беларусь
  - 🇨🇿 Чехия
- **Интернационализация (i18n)**: Интерфейс доступен на 5 языках: Русский, Украинский, Белорусский, Английский и Чешский.
- **PWA (Progressive Web App)**: Возможность установки на смартфон или компьютер, работа в оффлайн-режиме.
- **Высокая производительность**: Использование Web Workers для фонового парсинга и обработки данных, чтобы UI всегда оставался плавным.
- **Премиальный дизайн**: Современный интерфейс с эффектами Glassmorphism, адаптивной версткой и поддержкой мобильных клавиатур.

## Технологический стек

- **Core**: `React 19`
- **Build Tool**: `Vite 6`
- **Language**: `TypeScript 5`
- **Performance**: `Web Workers` для обработки логики парсинга.
- **PWA**: `vite-plugin-pwa` для кэширования и установки.
- **Testing**:
  - `Vitest` — для модульных тестов логики парсинга.
  - `Playwright` — для сквозных (E2E) тестов интерфейса.
- **Deployment**: `gh-pages`

## Запуск локально

### Требования
- `Node.js` 18.0.0 или выше
- `npm`

### Установка
```bash
npm install
```

### Разработка
Запуск dev-сервера с hot reload:
```bash
npm run dev
```

### Тестирование
Запуск модульных тестов (Vitest):
```bash
npm test
```

Запуск E2E тестов (Playwright):
```bash
npm run test:e2e
# Или с UI интерфейсом
npm run test:e2e:ui
```

### Сборка и деплой
Сборка production-версии:
```bash
npm run build
```

Публикация на GitHub Pages:
```bash
npm run deploy
```

```text
.

│   ├── utils/          # Утилиты, парсер номеров и Web Workers

├── tsconfig.json       # Конфигурация TypeScript
└── vite.config.ts      # Конфигурация Vite и PWA плагина
```

## Текущее состояние и качество
- **Тесты**: Логика парсинга покрыта модульными тестами на 100%. Основные сценарии использования проверены через E2E.
- **Производительность**: Минимальный размер бандла, оптимизированные перерендеры, вынос тяжелых вычислений в Worker.
- **UX**: Автоматический перевод в верхний регистр, поддержка ввода кириллицей/латиницей там, где это уместно.
